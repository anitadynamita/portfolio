Campaign

Siemens Xcelerator · 4 messages

Display advertising has a craft ceiling most people never see: file budgets measured in kilobytes, animation that must run identically across ad servers and browsers, and a design that has to survive its own constraints. For Siemens Xcelerator, I lived on both sides of that job.

01Objective

Give a B2B industrial-tech platform display advertising with real motion craft, inside ad-server weight budgets and cross-browser constraints that usually kill it.

02Idea

Design and code as one job: choreograph the motion in GSAP timelines rather than exporting flattened video, so the banner stays light, sharp and alive.

03Execution

Four message variants (at scale, competition, cybersecurity, ecosystem) built from scratch in HTML5: GSAP choreography, a Lottie-animated outro, retina-optimised assets, Flashtalking’s API handling the clickTag, one coded system, four stories.

04Result

Working software, not screen recordings, the banners below are the actual builds, still running. And the same visual grammar extended into vertical social film.
